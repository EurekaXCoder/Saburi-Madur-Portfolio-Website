Project Title: Real Estate Rent Prediction Using Linear Regression

Objective: Developed a predictive model to estimate house rents based on property features. Encoded categorical variables using appropriate techniques (e.g., one-hot encoding). Ensured the dataset was clean and properly formatted for modeling. Outcome: Delivered a high-performing predictive model, demonstrating strong data preprocessing and machine learning skills.
